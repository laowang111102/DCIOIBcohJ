# 项目概述 python676

本项目是一款基于Python语言的宿舍管理系统，采用Django框架进行后端开发，以MySQL数据库存储数据，前端使用Vue.js进行页面展示。

## 技术栈

- 后端：Python 3.x，Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

系统主要包括以下功能模块：

1. 学生信息管理
   - 学生基本信息录入
   - 学生信息查询与修改
   - 学生信息删除
2. 宿舍楼信息管理
   - 宿舍楼基本信息录入
   - 宿舍楼信息查询与修改
   - 宿舍楼信息删除
3. 宿舍信息管理
   - 宿舍基本信息录入
   - 宿舍信息查询与修改
   - 宿舍信息删除
4. 缴费管理
   - 缴费记录查询
   - 缴费信息录入
5. 报修管理
   - 报修申请
   - 报修进度查询

## 系统角色

- 管理员：负责系统管理与维护，以及宿舍信息的录入、查询、修改和删除等操作。
- 学生：可以查询个人信息、宿舍楼信息、宿舍信息，以及提交报修申请和缴费信息。

## 运行环境

- 后端：Python 3.x，Django，MySQL
- 前端：Vue.js，现代浏览器

## 部署步骤

1. 安装Python 3.x和Django
2. 安装MySQL数据库，创建数据库和表
3. 前端使用Vue.js搭建项目，安装依赖
4. 将后端代码部署到服务器
5. 将前端代码部署到Web服务器
